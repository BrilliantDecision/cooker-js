## Library for cookies in js

Example: https://next-js-cookie-api-example.vercel.app/

Use the library

```typescript
import { getCookie, setCookie } from "cookie-js-api";

/** setting a cookie */
setCookie('cookie-name', 'value')

/** getting a cookie */
getCookie('cookie-name')
```
